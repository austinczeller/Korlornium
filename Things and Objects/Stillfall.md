# [[Stillfall]]
## Description
A legendary plus two longbow used by [[Deepseer Poppy]]

It allows the user to teleport to a location the arrow lands once per long rest.

The wood is made of deep cobalt blue with dark purple knots in the wood. It is strung with [[Elf|Elven]] twine almost undetectable to the eye

It is now wielded by [[Thorne]]
